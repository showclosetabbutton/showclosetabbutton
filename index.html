How to show pocket button on each active tab?

Install Show Pocket Button browser extension.

The Show Pocket Button is an open-source MIT-licensed browser extension designed to shows a Save to Pocket button and a close tab button on every active tab on a web page. After installing it, you no longer have to search for the pocket button in layers of menus. You can save web pages in your pocket by just clicking a button. Show Pocket Button doesn't show ads and doesn't send user's data.
